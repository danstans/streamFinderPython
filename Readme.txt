This is a bot that allows users to subscribe to certain sports teams. The bot is called when /u/streamFinder is called in a comment. such calls can look like:

/u/streamFinder Athletics
/u/streamFinder athletics sf_giants
/u/streamFinder Oakland athletics sf_giants golden state warriors

The bot will add the commenter to a database and send them a PM of a sports stream link around gametime. All code written using python and SQLite
